# StudyPilot

StudyPilot is an instructor demonstration project for `10-152-117 Python Programming`.

The project shows how a useful study-planning tool can evolve from beginner-friendly Python fundamentals into more structured application architecture and, eventually, a web-application preview.

StudyPilot is not currently intended as a required student project. It is an instructor continuity project used to demonstrate how course concepts accumulate over time.

## Instructional Purpose

StudyPilot exists to demonstrate:

* how small Python concepts can grow into a useful application
* how variables, decisions, loops, functions, collections, files, testing, and structure connect
* how project development can be guided by clear intent and Refraction-Based Architecture thinking
* how the same project idea can be expressed in different architectural forms
* how AI-assisted development can be governed by human judgment, explanation, and review

## Project Versions

The `full` folder is planned to contain three completed reference versions:

* `01_console_app` - a polished plain-Python console version
* `02_architecture_preview` - a framework-free architecture preview with separated responsibilities
* `03_django_preview` - an optional instructor-facing Django/MVT preview

The `milestones` folder contains weekly instructional snapshots derived from the final project direction.

## Project Boundary

StudyPilot should remain instructional first.

It should be useful enough to matter, but not so abstract or framework-heavy that it becomes difficult to explain in the course context.

Students may inspect parts of it, but they are not expected to reproduce the full application unless the instructor later chooses to create a separate extension or advanced pathway.
