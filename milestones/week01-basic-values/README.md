# StudyPilot Week 1 - Basic Values

This milestone is a very small StudyPilot snapshot for Week 1.

It demonstrates:

* output with `print`
* assigned variables
* strings and numbers
* a simple expression to calculate remaining study time

## Run It

From the repository root:

```bash
python milestones/week01-basic-values/main.py
```

## What It Includes

The program displays a simple study plan summary using assigned values for:

* course name
* task name
* due date text
* estimated minutes
* completed minutes

It then calculates remaining minutes with a basic subtraction expression.

## What It Intentionally Omits

This Week 1 version does not include:

* input prompts
* conditionals
* loops
* functions
* collections
* file saving
* multi-file structure
